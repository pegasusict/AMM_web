# Audiophiles' Music Manager - AMM

__AMM__ is a music management system for very large collections.

It has the following processing capabilities:

* Importing / Parsing
* Fingerprinting
* Silence Trimming
* Converting
* Tag Retrieval
* Art Retrieval
* Lyrics Retrieval
* Tagging
* Renaming / Sorting
* exporting
